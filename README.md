## Getting Started

1. Run `npm install`
2. Run `npm run dev`

## Base SQL + disponibilites transport en temps reel

1. Creer un projet Supabase et renseigner `VITE_SUPABASE_URL` et `VITE_SUPABASE_ANON_KEY` dans `.env`.
2. Executer le script `sql/transport_inventory.sql` dans l'editeur SQL Supabase.
3. Ouvrir la page `transport` : les places disponibles sont lues depuis PostgreSQL.
4. Le bouton `Reserver` decremente les places via la fonction SQL `reserve_transport_seat`.
5. Les changements sont pushes en temps reel dans l'UI via Supabase Realtime.
